### Remote Code Executer

# Build With

- Nodejs
- Express
- Mongodb
- Mongoose
- Socketio
- React
- Bash

## Setup steps:

#### Get Started

```shell
# Clone repo
git clone <repo-url>
```

```shell
# cd Remote-Code-Executer-Realtime
```

```shell
# Go to client folder
cd client
npm install
cd ..
```

```shell
# Go to Server Folder
cd Server
npm install
cd ..
```

```shell
# Run bash file "start.sh"
bash start.sh
```
